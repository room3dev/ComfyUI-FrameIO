# ComfyUI-FrameIO

High-performance frame input/output nodes for AI video pipelines.

## Features
- Save batch IMAGE tensors to WebP
- Lossy or lossless WebP
- Skip identical frames via hashing
- STRING_LIST output (list-native)
- Optimized for long AI video generation

## Included Nodes
- Batch Save Image Sequence (WebP)

## Installation
```bash
cd ComfyUI/custom_nodes
git clone https://github.com/yourname/ComfyUI-FrameIO.git
```
